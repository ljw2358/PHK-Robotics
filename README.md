# HK Robotics

HK Robotics is a robotics-focused organization dedicated to building industrial-scale robotics systems.

## Scope
- Industrial Robotics
- Logistics / AMR
- Medical / Rehab Robotics
- Defense / Security Robotics
- Special-purpose Robots (disaster/plant)
- Humanoid Platforms

## Official Web App
This repository hosts the official Robotics-only web app frame for business registration and operations.
